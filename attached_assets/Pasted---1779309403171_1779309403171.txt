Проблемное заявление

    Вам предоставляется предварительно обученная модель спама электронной почты и ее требования. Ваша задача - развернуть модель и подготовить ее для конечного пользователя. Чтобы достичь этого, вам нужно разработать веб-страницу с использованием HTML и использовать данную модель, чтобы предсказать, является ли электронная почта спамом или нет. Веб-страница должна иметь простой удобный интерфейс.

    Короче говоря, вы должны разработать форму, которая принимает только один вход, текст электронной почты и возвращает, является ли электронная почта спамом или нет.

Установите репозиторий в редакторе кода:
GitHub Кодовое пространство

Кодовое пространство - это среда разработки, которая размещена в облаке. Каждое кодовое пространство, которое вы создаете, размещается GitHub в контейнере Docker, работающем на виртуальной машине. Вы можете выбрать из множества типов виртуальных машин, от 2 ядер, 8 ГБ оперативной памяти и 32 ГБ памяти, до 32 ядер, 64 ГБ оперативной памяти и 128 ГБ памяти.

Все личные учетные записи GitHub.com включают ежемесячную квоту бесплатного использования каждый месяц. GitHub предоставит пользователям в бесплатном плане 120 основных часов или 60 часов работы на 2-ядерном кодовом пространстве, плюс 15 ГБ памяти каждый месяц.

VScode [Факционал]

Visual Studio Code, также обычно называемый VS Code, представляет собой редактор исходного кода, созданный Microsoft с помощью Electron Framework, для Windows, Linux и macOS. Функции включают поддержку отладки, синтаксиса, интеллектуальное завершение кода, фрагменты, рефакторинг кода и встроенный Git.

Это очень мощный редактор кода и поддерживает несколько языков. Он имеет надежный набор дополнений к расширениям и является наиболее предпочтительным редактором кода для разработчиков.
